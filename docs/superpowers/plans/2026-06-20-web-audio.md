# Web Audio Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add synthesized keyboard effects and looping background music controlled by the existing sound button.

**Architecture:** A `useGameAudio` hook owns one lazily created `AudioContext`, short oscillator effects, and an interval-driven music sequence. `TypingGame` passes current paused, game-over, and mute state to the hook and calls its effect methods from existing input handling.

**Tech Stack:** React 19, TypeScript, Web Audio API, Vitest.

## Global Constraints

- Create audio only after the user's first interaction.
- No external audio files or packages.
- Correct input uses a bright short tone; incorrect input uses a lower warning tone.
- Background music loops at low volume, stops on pause/game over, and obeys the sound toggle.

### Task 2: Add the Web Audio hook

**Files:**
- Create: `hooks/use-game-audio.ts`

**Interfaces:**
- Consumes: `canPlayAudio` from `@/lib/game-audio`
- Produces: `unlockAudio(): void`, `playCorrectKey(): void`, `playWrongKey(): void`

- [ ] **Step 1: Implement lazy audio initialization and effects**

Create `useGameAudio({ soundOn, paused, gameOver })`. Store `AudioContext` in a ref. `unlockAudio` creates/resumes it only when allowed. Schedule a 0.08-second triangle-wave 880 Hz tone for a correct key and a 0.12-second sawtooth 180 Hz tone for an incorrect key. Each tone must use a gain envelope ending at zero.

- [ ] **Step 2: Add the background loop**

Use an effect that schedules one quiet sine-wave note every 500 ms through `[261.63, 329.63, 392, 329.63]` while `canPlayAudio` is true. Clean up its interval when muted, paused, game over, or unmounted.

- [ ] **Step 3: Verify the hook compiles**

Run: `pnpm test lib/game-audio.test.ts && pnpm build`

Expected: audio eligibility tests pass and the production build exits with status 0.
